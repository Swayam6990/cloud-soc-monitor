# Cloud Security Monitoring & Response System

A mini SIEM platform that ingests logs, detects threats,
supports analyst workflows, automated blocking,
behavior-based anomaly detection, dashboards,
and PDF reporting.

## Features
- Real-time log ingestion
- Rule & anomaly detection
- Severity classification
- Alert acknowledgment
- IP blocking simulation
- Visual analytics
- Analyst login
- Incident PDF export

## Tech Stack
Python, Flask, SQLite, Chart.js
